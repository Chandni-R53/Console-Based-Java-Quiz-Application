Console-Based Java Quiz Application

This is a console-based quiz application developed using Core Java.
The project allows users to attempt a quiz consisting of multiple-choice questions and receive a score at the end.

The application is designed to strengthen understanding of Java fundamentals, object-oriented programming (OOP) concepts, exception handling, and file handling.

Features
- Console-based user interaction
- Multiple-choice quiz questions
- Score calculation based on correct answers
- User input validation
- Quiz result storage using file handling

Technologies Used
- Java (Core Java)
- File I/O
- Object-Oriented Programming

Learning Outcomes
- Understanding of OOP concepts (classes, objects, inheritance, polymorphism,   abstraction)
- Practical use of exception handling
- File handling for data persistence
- Improved problem-solving and logical thinking
